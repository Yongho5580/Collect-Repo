## 프로젝트 설정 방법 및 실행 방법

1. 터미널에 ``npm install`` 을 입력해 프로젝트 실행에 필요한 패키지를 설치합니다.

2. 그 다음 터미널에 ``npm start`` 을 입력해 해당 프로젝트를 로컬 환경에서 실행합니다.
